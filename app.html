This project was relatively complex and it served as my introduction to web development.

The idea was simple, we'd make a form with html and collect data with it, and then display it again also using html and javascript.

At the time, we thought it'd be a challenge but it'd be doable.

After 6 months of work the end result was something like this:

Although it doesn't look like much, the product helped as achieve victory at Aerospace Regionals winning us a ticket to worlds.

Not my proudest work design wise but we did accomplish what we had set out to do.

Currently, I'm the only person working on improving it but I do have a few things planned.

The original scout app was server based where we'd upload data to out server, calculations would be done, the data would be stuffed into an sql table, and then it would be displayed.

The issues with that is that the server costs money.

The improvements I have in mind are then to either switch to firebase which is a free hosting server by google which comes with a database or hosting a website through bluetooth with a raspberry pi
