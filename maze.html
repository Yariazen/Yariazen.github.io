This was a project for advanced compsci to build a maze and solve it within the shortest time we can achieve

The final result was something like this:

Main class

In order to optimize the speed, I made my own amortized arraylist class rather than using the default arraylist class in java because although java's default arraylist works well for smaller lists, an amortized arraylist would be much faster once the list grows to above 100 elements.

I used recursive backtracking to build the maze and recursive backtracking to solve it.

In order to optimize those two algorithims, I made my own linkedlist rather than using arraylist in order to reduce list search time to O(1) time rather than O(log(n)) time.

Finally, I used Stanford's standard draw library to draw the maze.
